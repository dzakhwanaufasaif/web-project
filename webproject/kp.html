Kebijakan Privasi

Pengumpulan Data

pengumpulan data atas nama pasien, nomor identitas dijamin aman dari pembajakan seperti penyalah gunaan data pribadi pasien di klinik ini, dan juga data administrasi seperti alamat, nomor telepon dan email.

Penyimpanan dan Keamanan Data

Akses terbatas, tidak semuua pegawai dapat melihat data pasien, seperti dokter yang hanya bisa melihat data pasien nya saja, Keamanan Sistem seperti melindungi dari serangan digital contohnya server ditempatkan di tempat yang aman.

Penggunaan dan Pembagian Data

Tujuan internal data digunakan hanya untuk kepentingan pelayanan kesehatan tidak untuk konsultasi yang berlebihan contohnya dokter menggunakan data untuk diagnosis dan perawatan, dan juga data dipakai untuk penelitian medis tetapi sudah di ananomi kan atas persetujuan pasien, jika pasien setuju

Kesimpulan

Pengelolaan data pasien harus dilakukan secara aman, transparan, dan sesuai hukum, dengan menjamin keamanan sistem, kerahasiaan data, serta hak pasienagar kepercayaan dan etika pelayanan kesehatan tetap terjaga.
