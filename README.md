# DESKRIPSI WEBSITE
Web Klinik Sahabat dirancang dengan konsep profesional, bersih, dan modern, yang bertujuan untuk memberikan kesan kepercayaan serta kemudahan akses informasi bagi calon pasien.

1. Identitas Visual & Estetika
Palet Warna: Menggunakan kombinasi dominan Biru dan Putih, yang secara psikologis melambangkan kepercayaan, kebersihan, dan profesionalisme di bidang medis.
Tipografi: Menggunakan font sans-serif (seperti Montserrat atau Segoe UI) yang tegas namun ramah, memberikan kesan modern dan mudah dibaca oleh berbagai kalangan usia.
Bentuk Elemen: Penggunaan sudut-sudut yang sangat membulat (extra rounded corners) pada kotak informasi dan tombol memberikan kesan user-friendly dan tidak kaku.

2. Struktur Navigasi (Header)
Halaman ini memiliki navbar tetap di bagian atas dengan logo di sisi kiri dan menu navigasi di sisi kanan:
Menu: Terdiri dari Home, Layanan (dengan dropdown), Tentang Kami, S&K (Syarat & Ketentuan), dan Kamus.

3. Responsivitas
Mobile-Friendly: Dibangun dengan framework Bootstrap, web ini bersifat responsif. Pada perangkat seluler (HP), tata letak dua kolom akan otomatis bertumpuk secara vertikal agar tetap nyaman dibaca tanpa perlu memperbesar layar (zoom).
